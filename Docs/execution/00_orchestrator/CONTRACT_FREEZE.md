# Contract Freeze

The following contracts are frozen and require A0 approval to modify:

- Command bus types (`core/commands/types.ts`)
- Command result/error model
- Document history transaction shape (`core/document-history/types.ts`)
- Save/export/session action taxonomy (`core/session/types.ts`)
- Search result shape (`core/search/types.ts`)
- Review metadata shape (`core/review/types.ts`)
- Macro run report shape
- Shared UI feedback contract
