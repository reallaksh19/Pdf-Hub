# Manual Smoke - Chrome and Edge

## Core Flows
- Open file, save working doc, export flattened review copy, download processed PDF.
- Rotate/extract/split/delete/replace from toolbar and thumbnail context menu.
- Undo/redo for annotation and document mutation flows.
- Search: find, next/prev, scroll-to-hit, active-hit persistence.
- View modes: continuous/single/two-page and fit modes on resize.
- Text-mark and annotation interactions by tool contract.
- Macro run with validation, logs, and output queue.

## Pass Criteria
- No crashes, no detached-buffer errors, no silent failures.
- Errors are visible and recoverable.
- Keyboard and focus flows remain usable.
